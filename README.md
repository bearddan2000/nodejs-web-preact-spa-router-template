# nodejs-web-preact-spa-router-template

## Description
A template for the preact framework, a superset for react,
on a single page application.

Feature
- router

## Tech stack
- npm
- preact

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
